# Ninho Baby SC 🐻💛

A modern and responsive institutional website developed for **Ninho Baby SC**, a baby clothing store specialized in fashion for children from 0 to 3 years old.  
This project was created with a warm visual identity, smooth animations, and a pleasant user experience in mind.

---

## ✨ Preview

This website includes:

- Animated hero section
- Modern fixed navbar
- Brand presentation sections
- Product category cards
- “Why Choose Us?” section
- Shopee call-to-action section
- Custom footer
- Scroll reveal animations
- Fully responsive layout

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts
